ID Mapping
=============================
Virtual Socket Manager counter is assigned ID 0
  Reconfigurable Module count_up is assigned ID 0
  Reconfigurable Module count_down is assigned ID 1
  Reconfigurable Module count_half is assigned ID 2


Address Structure
=============================
The addresses are structured as follows:
  There is no VSM Select slice as there's only one VSM
  address[ 7: 6]  : Selects the Register Bank
  address[ 5: 2]  : Selects the Register within the bank
